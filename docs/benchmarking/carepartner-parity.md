# 케어파트너 ↔ 일손 기능 패리티 매트릭스

> 케어파트너(carepartner.kr) 대비 일손 기능/화면 비교 + 작업 트래킹.
> 케어파트너 코드는 비공개 → 라이브 서비스 관찰 기반.
> 차이 유형: 동일 / 방식 다름 / 일손 미구현 / 일손 고유
