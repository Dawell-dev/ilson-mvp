## 💡 어떻게 활용하나

위 5개 메모리가 등록되면, Claude는 새 대화 시작 시 자동으로:
- 대표님이 누군지 알고 시작
- 더불어그룹 프로젝트 구조 알고 시작
- 김주은이 PM임을 알고 기술 디테일은 PM에게 위임할 것을 안내
- 비즈니스 의사결정 중심으로 답변 스타일 조정

**활용 예시**:
- "다웰솔루션 5월 청구 결과 검토해줘" → 컨텍스트 즉시 이해
- "일손 채용 플랫폼 시장성 분석해줘" → 프로젝트 배경 파악된 상태
- "운영팀 인력 충원 시점 의견 줘" → 박선희 합류 진행 중 알고 답변

## ❓ 자주 묻는 질문

**Q. 메모리가 30개 한계 있다는데?**
A. 위 5개는 영구 기본 정보. 추가 메모리는 대표님 판단대로 등록 가능. 단, 일시적 정보(특정 PR 상태 등)는 등록 비추천 (작업 종료 후 stale).

**Q. 다웰솔루션 / 일손 메모리 분리는?**
A. Claude의 Project 기능 사용 권장. 다웰솔루션 Project / 일손 Project 각각 만들면 메모리 자동 분리.

**Q. 김주은이 등록한 메모리랑 안 섞이나?**
A. 안 섞입니다. Claude 메모리는 각 사용자 계정별 분리. 대표님이 등록한 메모리는 대표님 대화에만 적용.
